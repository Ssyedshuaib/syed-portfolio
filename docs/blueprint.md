# **App Name**: Founder-Craft

## Core Features:

- Orbital Interactive Hero: A high-end animated hero section featuring a central 'Build' node with orbiting nodes (Learn, Launch, Scale) that react to mouse movements.
- DevNexus Showcase Module: An immersive flagship project section utilizing parallax scrolling and glassmorphic device frames to highlight DevNexus as the centerpiece.
- Project Bento Navigator: A luxury bento grid layout to browse side projects like Reverie and NovaPU with staggered entrance animations.
- Philosophy Insight AI: An AI-powered tool that allows visitors to query the builder's specific product philosophies and how they were applied in DevNexus.
- Interactive Founder Journey: A scroll-linked vertical timeline illustrating the development career from CSE student to Product Builder.
- Live Impact Dashboard: Animated statistic counters visualizing growth metrics, products designed, and project timelines for social proof.
- Premium Connectivity Hub: A minimalist, floating contact area with magnetic buttons for quick access to GitHub, LinkedIn, and email.

## Style Guidelines:

- Primary Color: #D7B29D (Rose-gold Sand) for a high-end, founder-focused look.
- Background: Deep Obsidian (#0A0705) with Secondary Layering (#14100D) and Card Surfaces (#1B1512).
- Headline font: 'Inter Tight'. Body font: 'Inter'. (Note: currently only Google Fonts are supported.)
- Generous white space (breathing room) with a product-first hierarchy, leveraging Bento grids and large hero visuals.
- Luxury transitions: Smooth scroll-trigger reveals, staggered card entrances, magnetic buttons, and mouse-parallax interactions.
- Elegant, ultra-minimal thin-stroke icons that align with the sophisticated aesthetic of brands like Linear or Apple.
